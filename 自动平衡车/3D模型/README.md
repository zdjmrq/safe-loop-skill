## 快速使用

1. 装 [OpenSCAD](https://openscad.org/) → 打开.scad → F6渲染 → 导出STL
2. 或 [openscad.cloud](http://openscad.cloud/) 在线导出
3. STL → Cura/PrusaSlicer → 切片 → 打印

## 重心调节指南

自平衡车的核心：**重心必须在轮轴正上方**。

```
调参方法:
  pitch_servo_x  ↑ → 平台前移 → 重心前移 → 车往前倾
  pitch_servo_x  ↓ → 平台后移 → 重心后移 → 车往后仰
  arm_offset_x   ↓ → 平台后移 (更负) → 同上

  建议: 装好后先不上平台, 只装底盘+电池+电机+轮子
        找到自然平衡点, 再根据前倾/后仰调整pitch_servo_x
```

## ⚠️ 安装注意事项

1. **电机座**: N20减速箱端塞进矩形腔, 电机体进圆孔段, 别装反
2. **螺丝长度**: 底板上方的螺丝不要超过8mm (M3×6最安全), 否则会顶到下面的电池
3. **舵机螺母**: Pitch舵机的4颗M3螺母嵌在底盘底面的六角槽里, 螺丝从上往下拧
4. **重心**: 全部装好后, 提起来看车身是否水平。不平就调pitch_servo_x
5. **电机座镜像**: 第二个在切片软件里右键→镜像
