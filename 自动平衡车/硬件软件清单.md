### 4.2 电源线

```
LiPo 11.1V (+)
    │
    ├──[船型开关]──┬── DRV8833 VM 引脚（电机电源）
    │               │
    │               ├── LM2596 IN+ ──→ LM2596 OUT+ (5V)
    │               │                    │
    │               │              ┌─────┼─────┬──────────┐
    │               │              │     │     │          │
    │               │           STM32  HC-05 舵机×2    AMS1117
    │               │           5V pin  VCC   V+      IN ─→ OUT (3.3V)
    │               │              │     │     │          │    │
    │               │              │     │  [1000μF]      │  MPU6050×2
    │               │              │     │   ─┴─ GND      │   VCC
    │               │              │     │                │
    │               └──────────────┴─────┴────────────────┴─── 共地 GND
    │
LiPo 11.1V (-) ──[开关]────────────────────────────────────── 共地 GND
```

> **要点**：
> - 所有模块的 **GND 必须连在一起**（共地），否则信号无法参考
> - 1000μF 电容正极接 5V、负极接 GND，物理上靠近舵机电源引脚
> - 先焊电源部分，万用表确认电压正确了再接信号线

## 五、两个 MPU6050 的地址设置

GY-521 模块默认 I2C 地址为 **0x68**。挂两个时需要区分：

| IMU | AD0 引脚状态 | I2C 地址 | 操作方法 |
|-----|:---:|:---:|------|
| 车体（下部） | GND (默认) | **0x68** | 不动，保持出厂状态 |
| 平台（上部） | VCC | **0x69** | 把模块上的 AD0 焊盘用锡连到旁边的 VCC 焊盘 |

GY-521 模块上有 AD0 标识的焊盘，旁边一般就有 3.3V 的过孔。用烙铁点一滴锡连通即可。

## 六、HC-05 蓝牙模块配置

### 6.1 进入 AT 模式

1. HC-05 的 EN/KEY 引脚接高电平（3.3V）上电
2. 模块灯慢闪（约 2 秒一次）→ 已进入 AT 模式
3. 串口波特率固定为 **38400**

### 6.2 配置指令

```
AT                    # 测试通信，应返回 OK
AT+NAME=BalanceCar    # 设置蓝牙名称
AT+UART=115200,0,0    # 设置通信波特率 115200（和 STM32 一致）
AT+ROLE=0             # 设为从机模式
```

> 这些只需要配一次，HC-05 会记住。之后的正常使用中，EN 脚悬空或接 GND 上电，波特率就是你设的 115200。代码里我会写好——对你来说就是"手机搜到 BalanceCar 这个蓝牙→连接→打开串口 App→发指令"。

## 七、采购前检查清单

下单前逐项确认：

- [ ] STM32F411CEU6 是 "BlackPill" 黑色小板，USB-C 口
- [ ] N20 电机 **带 AB 相霍尔编码器**（排线 6 针，不是 2 针）
- [ ] 轮子联轴器内径匹配 N20 电机轴（拍照问卖家确认）
- [ ] DRV8833 是小红板模块（不是裸芯片）
- [ ] MG996R **金属齿轮**版（有塑料齿轮版也叫 MG996，易坏）
- [ ] LM2596 是 **5A** 版本（商品标题或参数里写 output 5A）
- [ ] HC-05 是 **3.3V** 版本带底板（带 6 个排针 + EN 引脚）
- [ ] LiPo 电池是否有充电器（没有的话加一个 B6 充电器）
- [ ] 两个 GY-521 模块（别只买一个！）
- [ ] 杜邦线公母各一包（不够很头疼）
- [ ] 1000μF 16V 电解电容（别漏了这个，2 个）
- [ ] 螺丝刀 + 万用表 + 烙铁是否已有
